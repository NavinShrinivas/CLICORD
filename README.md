# CLICORD 

CLICORD is a cli based chat system written from scratch using websockets in go.

## Features 

### Ping pong server 

```
   in /client : $ go run *.go ping 
   in /server : $ go run *.go
```
### Creating a particular server/lobby for clients

```
   in /client: $ go run *.go lobby server_name
   in /server: $ go run *.go
```
